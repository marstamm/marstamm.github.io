GsSqueak

This project emerged from a bachelor's project at the HPI Software Architechture Group in the year 2017/18. From the abstract of 'Solving Language Differences of Squeak and GemStone Smalltalk on a Method Level':

Tooling for databases is hard to develop. GemStone/S offers the possibility to execute arbitrary Smalltalk code inside an object database. Squeak/Smalltalk provides powerful tooling and adaptive programming. With GsSqueak we want to integrate the Squeak programming environment into GemStone.

EasyWash Website

This Website offers an open-source alternative to the closed-source app by Schneidereit GmbH. It can be used to check on the status of washing machienes in the dorms.

It is written in NodeJS and currently runs on AWS lambda.
